About Suppa-Chat

Welcome to Suppa-Chat!

Suppa-Chat is a modern, real-time chat application that allows you to connect with friends and family instantly. Built with cutting-edge technology, our app provides a seamless messaging experience with a clean, intuitive interface.

Features

  • Real-time messaging with instant delivery
  • Voice message recording and sharing
  • Emoji picker for expressive conversations
  • Typing indicators to see when others are typing
  • Desktop notifications for new messages
  • Responsive design that works on all devices
  • Secure Firebase backend for reliable performance
  • Reply to specific messages
  • React to messages with emojis

How to Use

Simply type your message in the input field and press Enter or click the send button. Use the microphone icon to record voice messages, and the emoji button to add emojis to your messages. When someone is typing, you'll see a typing indicator below their name. To reply to a message, long press on it and select "Reply". To react to a message, long press and select "React".
